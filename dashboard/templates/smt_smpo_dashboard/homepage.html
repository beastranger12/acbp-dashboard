About

SMT-SMPO-Dashboard is a Django and Bootstrap powered web site created using the Django Octane template. Right now it is still under construction but soon it will do some awesome things.

Server Side Engine

Django is used as the server side engine for powering this website. As is explained on the homepage for Django, Django is a high-level Python Web framework that encourages rapid development and clean, pragmatic design. Visit the Django site to learn more Django.

Client Side

Many components are used to implement the client-side functionality. Bootstrap is the main CSS/JS library used for rendering pages of the site. The packages shipping Bootstrap and its dependencies are from NPM packages. Visit the mentioned links to learn more.
